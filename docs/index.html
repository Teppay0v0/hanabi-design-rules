Claude Code でデザインを作るとき「なんかAIっぽい」「詰まってる」「カラフルすぎる」を防ぐスキルです。インストールしておくだけで、LP・HP・スライドの出力が変わります。根本の考え方は、引くこと。

配色を選ぶ → ワンコマンドでインストール → Before / After を見る → GitHub →

ワンコマンドでインストール

ターミナルにこれを貼るだけです。~/.claude/skills/ に配置されます。

curl -fsSL https://raw.githubusercontent.com/Teppay0v0/hanabi-design-rules/main/install.sh | bash

更新したいときも、同じコマンドをもう一度実行するだけです。アンインストールは rm -rf ~/.claude/skills/hanabi-design-rules。

USAGE

インストール後は、Claude Code に普通に話しかけるだけ。「このサービスのLPを作って」「会社紹介のスライドを作って」— Claude が自動でこのスキルを読み込み、ルールに沿ったデザインを出力します。

何が変わるのか

AIのデフォルトでやりがちな表現を禁止し、代わりの手段を指定します。

禁止代わりに
絵文字モノラインSVGアイコン
イタリックサイズ・ウェイト・字間で強調
高彩度のベタ塗り背景・グラデーション白＋メインの極薄ティント（wash）
ドロップシャドウ細罫で奥行きを表現
角丸カード直角のみ
3色以上の主張する配色メイン＋サブの2色（＋インク・グレー）
大型CTAボタンテキストリンクのみ
ストック写真タイポグラフィ・数字・線で構成
